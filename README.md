# Yamaha-Reface-CP-Assistant
A lightweight, web-based MIDI tool designed for the Yamaha Reface CP, featuring real-time TSD (Tonic/Subdominant/Dominant) chord function visualization, seamless pitch transposition, and smart auto-pedal capabilities. 基于 Web 的轻量级智能 MIDI 辅助工具，专为雅马哈 Reface CP 设计，无需安装驱动即可在浏览器中提供实时的 TSD 和弦功能可视化、无缝升降调以及智能自动踏板功能，极大地拓展演奏与乐理学习体验。
